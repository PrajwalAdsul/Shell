# Shell Implementation in C language

## Run :-
 $ make <br/>
 $ ./shell 
  
## Features implemented :-  
    Read commands in a loop,
    show prompt,
    run commands,
    run commands in background,
     > (output redirection),
    < (input redirection) and
    | (pipe)
    Handle signals Ctrl-C and Ctrl-z
    fg , bg
